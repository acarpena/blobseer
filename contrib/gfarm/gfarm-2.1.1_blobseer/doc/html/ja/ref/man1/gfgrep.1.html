gfgrep

Name

gfgrep -- 並列ファイルを grep する

Synopsis

gfrun {gfarm:gfgrep} -o 出力gfarm-URL 正規表現 入力gfarm-URL...

gfrun {gfarm:gfgrep} -o 出力gfarm-URL -e 正規表現 入力gfarm-URL...

DESCRIPTION

gfrun から呼び出されるアプリケーションのサンプル・プログラムです。

また、ローカルファイルビューを利用して Gfarm ファイルに出力する例にも なっています。

入力 gfarm-URL から正規表現を検索し、結果を 出力 gfarm-URL に出力し ます。

ソースを見ると分かる通り、gfrun から呼び出されるアプリケーションは、 -I および -N オプションを解釈し、ここから得た値で gfs_pio_set_local() を呼び出す必要があります。

OPTIONS

-e 正規表現

「-」で始まる正規表現を指定したい場合に用います。

-o gfarm-URL

出力 Gfarm ファイルの URL を指定します。

-?

引数オプションを表示します。
