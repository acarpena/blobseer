gfrep

Name

gfrep -- Gfarm ファイルの複製を作成する．

Synopsis

gfrep -H ホスト指定ファイル [-vm] gfarm-URL...

gfrep -P ホスト対指定ファイル [-vm] gfarm-URL...

gfrep -d 複製先ホスト [-s 複製元ホスト] [-I ファイルセクション] [-vm] gfarm-URL...

gfrep [-D 複製先ドメイン名] [-S 複製元ドメイン名 | -s 複製元ホスト] [-N 複製数] [-v] gfarm-URL...

DESCRIPTION

引数 Gfarm-URL で指定したファイルの複製を， 指定されたホストに作成します．

Gfarm-URL がディレクトリの場合， そのディレクトリを再帰的に複製します．

OPTIONS

-D 複製先ドメイン名

ファイルの複製を作成する先のホストのドメイン名を指定します． このオプションは単独で，あるいは，-S や -s オプションと共に用います．

このオプションも -d オプションも指定しない場合， メタデータに登録された全ホストのいずれかに複製を作成します． -N オプションの項も参照してください．

-d 複製先ホスト

複製作成先のホスト名を1つだけ指定します． このオプションは単独で，あるいは，-s オプションと共に用います．

このオプションも -D オプションも指定しない場合， メタデータに登録された全ホストのいずれかに複製を作成します．

-H ホスト指定ファイル

各フラグメントの複製を作成するホスト名のリストを記述した ファイル名を指定します． ホスト名は，ファイル中の各行に一ホストずつ記述します． 指定したホスト名の数がフラグメントの数よりも少ない場合， フラグメントの数と一致するまでホスト名のリストを繰り返し指定した ものと見なします．

ファイル名として ``-'' を指定した場合，標準入力からホス ト名のリストを読み込みます．

-I ファイルセクション

このオプションで指定した名前のセクションのみ，複製を作成します． 現在のところ，このオプションは，-d オプションを指定した 場合にしか動作しません．

このオプションは省略できます．

-m

複製ではなく移動を行ないます． 現在のところ，このオプションは，-H，-P, -d オプションを指定した場合にしか動作しません．

-N 複製数

必要な複製の数を指定します． 複製先に，指定された数以上の複製が既に存在する場合は， 複製の作成も削除も行いません． このオプションは単独で，あるいは -D， -S，-s オプションと共に用います．

このオプションを省略すると -N 1 と指定された ものとみなされます．

-S オプションあるいは -s オプションで指定された複製元のホストは，複製を作成する対象ホストから 除外されます． つまり，-S オプションあるいは -s オプションで指定された以外のホストに，-N で指定した 数以上の複製が存在することを保証します． ただし，複製対象のホストが，複製元のホストの部分集合である 場合には除外されません．複製対象が存在しなくなってしまうからです． 例えば「-N 3 -S subdomain.example.org -D example.org」と指定して実行すると， 結果として，example.org に属すが subdomain.example.org には属さないホストに， 3つ以上の複製が存在することが保証されます．

-P ホスト対ファイル

複製元のホストと複製先のホストの対のリストを記述した ファイル名を指定します． ファイルの各行には，複製元ホストと複製先ホストを，この順で空白で 区切って指定します． 指定した Gfarm-URL を構成するファイルが，複製元ホストに存在する場合のみ， そのファイルを複製先ホストにコピーします．

-S 複製元ドメイン名

ファイル複製元のホストのドメイン名を指定します． このオプションは単独で，あるいは，-D オプションと共に用います．

このオプションも -s オプションも指定しない場合， 複製元のホストは，全ノードから選ばれます．

また，このオプションで指定されたホストは，複製を作成する対象ホスト から除外されます． 詳しくは -N オプションの項を参照してください．

-s 複製元ホスト

ファイル複製元のホスト名を指定します． このオプションは単独で，あるいは，-D や -d オプションと共に用います．

このオプションも -s オプションも指定しない場合， 複製元のホストは，全ノードから選ばれます．

また，このオプションで指定されたホストは，複製を作成する対象ホスト から除外されます． 詳しくは -N オプションの項を参照してください．

-v

複製処理に関して，冗長な表示を行ないます，

-?

引数オプションを表示します．
