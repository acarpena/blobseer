gfexport

Name

gfexport -- Gfarm ファイルを通常のファイルに変換する

Synopsis

gfexport [-H ホスト指定ファイル] [-I フラグメント番号] gfarm-URL

DESCRIPTION

指定した Gfarm URL の表す Gfarm 並列ファイルを、通常のファイルに変換 して標準出力に出力します。

OPTIONS

-H ホスト指定ファイル

フラグメントを取り出すホスト名のリストを指定します。 ホスト名は、各行に一ホストずつ記述します。 省略した場合には、自動的に選ばれたホストからフラグメントを得ます。

ファイル名として ``-'' を指定した場合、標準入力からホスト名のリストを 読み込みます。

-I フラグメント番号

並列ファイル中の、指定したフラグメントのみを取り出します。 実行形式のファイルの場合には、このオプションで実行アーキテクチャ (例 「sparc-sun-solaris8」) を指定することによって、指定したアーキ テクチャの実行ファイルを取り出せます。

-?

引数オプションを表示します。
