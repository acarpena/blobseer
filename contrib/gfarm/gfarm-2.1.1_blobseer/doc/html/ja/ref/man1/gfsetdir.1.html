gfsetdir

Name

gfsetdir -- Gfarm 作業ディレクトリを変更するコマンド列を出力する。

Synopsis

gfsetdir [-s | -c] [ディレクトリ]

DESCRIPTION

Gfarm 作業ディレクトリを 指定したディレクトリに変更するシェルコ マンドを標準出力に出力します。ディレクトリを指定しない場合は、ユーザの Gfarm ホームディレクトリである gfarm:/グローバルユーザ に変更する コマンドを出力します。

ユーザが使用しているシェルを自動的に判定してそのシェルが実行可能な コマンドを出力します。

出力コマンドを実行して Gfarm 作業ディレクトリを変更する gfcd を利用するために、各ユーザのシェル設定ファイルに以下の定義を追加して ください。

Bourne shell 系シェルの場合
gfcd() { eval "`gfsetdir $1`"; }
csh 系シェルの場合
alias gfcd 'eval `gfsetdir \!*`'

OPTIONS

-s

Bourne shell 系シェル用のコマンドを出力します。

-c

csh 系シェル用のコマンドを出力します。

-?

引数オプションを表示します。

SEE ALSO

gfcd(1), gfpwd(1)
