gfwhere

Name

gfwhere -- Gfarm ファイルのレプリカの所在を表示する。

Synopsis

gfwhere [-I フラグメント番号] [-s] gfarm-URL...

DESCRIPTION

指定した Gfarm-URL を構成するファイルについて、レプリカの所在を表示 します。

フラグメント化されたファイルの場合、各行は、下記の形式になります。

フラグメント番号: レプリカのあるホスト名....

また、実行ファイルの場合、各行は、下記の形式になります。

アーキテクチャ: レプリカのあるホスト名....

OPTIONS

-I フラグメント番号

レプリカの所在を表示するフラグメントを指定します。 実行形式のファイルの場合には、このオプションで実行アーキ テクチャ(例 「sparc-sun-solaris8」) を指定することによっ て、指定したアーキテクチャのレプリカの所在を表示できます。

-s

ファイル・フラグメントのサイズも表示します。

-?

引数オプションを表示します。
