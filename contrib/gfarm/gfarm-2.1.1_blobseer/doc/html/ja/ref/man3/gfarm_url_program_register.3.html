gfarm_url_program_register

Name

gfarm_url_program_register -- プログラムを登録する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_url_program_register(char *gfarm_url, char *architecture, char *filename, int nreplicas);

DESCRIPTION

引数 architecture で指定した機種向けの実行形式ファイル filename を、 gfarm_url で指定したパス名に登録します。 また、 nreplicas に、作成するレプリカの数を指定します。

引数 nreplicas は、将来のバージョンで削除される予定です。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

GFARM_ERR_OPERATION_NOT_PERMITTED

指定したファイルに、実行可能ビットが立っていませんでした。

その他

上記以外のエラーが生じたことを示します。
