gfarm_initialize

Name

gfarm_initialize -- Gfarm ライブラリを初期化する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_initialize(int * argcp, char *** argvp);

DESCRIPTION

Gfarm ライブラリを、Gfarm アプリケーション用に初期化します。 引数には、プログラムの main 関数の argc 引数へのポインタと、 argv 引数へのポインタを渡します。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。
