gfarm_stringlist_length

Name

gfarm_stringlist_length -- stringlist の文字列数を返す

Synopsis

#include <gfarm/gfarm.h>

int gfarm_stringlist_length(gfarm_stringlist * listp);

DESCRIPTION

listp の表す文字列配列の配列要素数を返します。

RETURN VALUES

文字列の数を返します。
