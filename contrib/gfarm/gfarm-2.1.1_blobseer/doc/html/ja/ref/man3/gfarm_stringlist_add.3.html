gfarm_stringlist_add

Name

gfarm_stringlist_add -- stringlist に文字列を追加する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_stringlist_add(gfarm_stringlist * listp, char * s);

DESCRIPTION

listpの表す文字列配列の末尾に、文字列 s を追加します。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

その他

上記以外のエラーが生じたことを示します。
