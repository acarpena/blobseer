gfarm_url_section_replicate_to

Name

gfarm_url_section_replicate_to -- ファイルセクションのレプリカを作成する

Synopsis

#include <gfarm/gfarm.h>

char *gfarm_url_section_replicate_to(const char *gfarm_url, char *section, char *dsthost);

DESCRIPTION

ファイル gfarm_url のファイルセクション section の複製を、 dsthost に作成します。

ファイルセクションとは、実行形式のファイルの場合はアーキテクチャ名、 通常ファイルの場合はファイルインデックス番号となります。 ファイルインデックス番号を整数ではなく、文字列 section で指定する必要があることに注意してください。また、またこの文字列は、 余分な空白や、先行 0 を含んではいけません。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

GFARM_ERR_NO_SUCH_OBJECT

指定されたファイルが存在しません。

GFARM_ERR_TEXT_FILE_BUSY

指定されたファイルが現在（他のプロセスにより）更新中のため、 ファイルの複製が作成できません。

その他

上記以外のエラーが生じたことを示します。
