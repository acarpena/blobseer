gfs_pio_ungetc

Name

gfs_pio_ungetc -- 読み込んだ文字を戻す

Synopsis

#include <gfarm/gfarm.h>

int gfs_pio_ungetc(GFS_File gf, int c);

DESCRIPTION

ファイル gf から読み込んだ文字 c を、読み込まなかったことにします。

RETURN VALUES

EOF

gfs_pio_ungetc() は、関数 gfs_pio_getc(3) の呼びだしの直後にしか 呼ぶことができません。この制約に反した場合、 EOF をエラーとして返します。

その他

正常に処理が完了したことを示します。
