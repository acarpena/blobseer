gfs_opendir

Name

gfs_opendir -- ディレクトリのオープン

Synopsis

#include <gfarm/gfarm.h>

char *gfs_opendir(const char * gfarm_url, GFS_Dir * dirp);

DESCRIPTION

gfarm_url で指定したディレクトリをオープンします。

RETURN VALUES

NULL

正常終了を示します。

GFARM_ERR_NO_MEMORY

メモリ不足のため処理を中止しました。

GFARM_ERR_AUTHENTICATION

gfmd(8) に対するユーザー認証が失敗しました。

GFARM_ERR_PERMISSION_DENIED

gfarm_url へアクセスする権限が、ありません。

GFARM_ERR_NO_SUCH_OBJECT

指定されたディレクトリが存在しません。

その他

上記以外のエラーが生じたことを示します。

SEE ALSO

gfs_closedir(3), gfs_readdir(3)
