gfdf

Name

gfdf -- display gfarm filesystem disk free space

Synopsis

gfdf -H hostfile [-i]

gfdf -h hostname... [-i]

DESCRIPTION

gfdf displays disk usage and free space of specified gfarm filesystem node.

OPTIONS

-H hostfile

Specifies a list of filesystem nodes. The hostfile consists of the hostname of a filesystem node in each line.

If ``-'' is specified, standard input is used to read the host list.

-h hostname

Specifies the hostname of a filesystem node. This option can be used multple times.

-i

Displays i-node information instead of disk space information.

-?

Displays a list of command options.
